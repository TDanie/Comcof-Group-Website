# Comcof Group — Website

The Commerce of Coffee. Corporate website for Comcof Group, live at [comcofgroup.com](https://comcofgroup.com).

Static multi-page site (clean URLs via `vercel.json`), deployed on Vercel. Pages: home, about, group, coffee, global, responsibility, partners, insights, contact. Shared styles in `assets/style.css`, behaviour in `assets/site.js`, world map geometry (Natural Earth 110m, public domain) in `assets/world-land.js`.

## Deploy

```
vercel --prod
```
